Products

This is a sample of our expertise.

Each of these products is under Apache 2 license, so feel free to try and use them as you need. Just call us if you need services or have some questions.
All the products together forms a complete reporting and analysis solution.

simplypdf-core

This java library allows you to create a valid PDF 1.7.
It wraps the PDF format in a simple way that allows you to concentrate on the content to communicate rather than the code to implement.

simplypdf-layout

This java library extends functionalities of simplypdf-core.
It adds layout possibilities and introduce widgets (table of content, datatable, headers/footers, footnotes, etc.) to ease the use of the low-level API.

simplypdf-designer

This application allows to design a PDF, no more by coding it, but by using a simple to use and still efficient editor.
You can then concentrate on style, appearance and content.
It also greatly improve your productivity!
